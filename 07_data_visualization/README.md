# 📊 Data Visualization

This module covers the fundamentals of data visualization using Python libraries such as Matplotlib and Seaborn.

## 🎯 Learning Objectives

- Understand the importance of data visualization
- Create different types of charts
- Explore trends and patterns in datasets
- Prepare data for Machine Learning analysis

## 📚 Topics Covered

- Matplotlib Basics
- Line Charts
- Bar Charts
- Pie Charts
- Histograms
- Scatter Plots
- Seaborn Visualizations

## 📁 Files

| File | Description |
|--------|-------------|
| matplotlib_intro.ipynb | Introduction to Matplotlib |
| line_chart.ipynb | Line chart examples |
| bar_chart.ipynb | Bar chart examples |
| pie_chart.ipynb | Pie chart examples |
| histogram.ipynb | Histogram visualization |
| scatter_plot.ipynb | Scatter plot examples |
| seaborn_basics.ipynb | Introduction to Seaborn |
