# 📂 File Handling in Python

This module covers file operations in Python, including reading, writing, and updating text files.

## 🎯 Learning Objectives

- Understand file handling concepts
- Read data from files
- Write data to files
- Append new information to existing files
- Store simple records using text files

## 📁 Files

| File | Description |
|--------|-------------|
| read_file.py | Read data from a text file |
| write_file.py | Write data to a file |
| append_file.py | Append data to an existing file |
| student_records.txt | Sample student records |
| expense_tracker.py | Mini expense tracking project |

## 💡 Skills Learned

- File I/O Operations
- Data Persistence
- Text File Management
- Basic Record Keeping
