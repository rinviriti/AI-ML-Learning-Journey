# 🧠 Deep Learning

This module focuses on neural networks and deep learning using TensorFlow and Keras.

## 🎯 Learning Objectives

- Understand Artificial Neural Networks (ANN)
- Learn Deep Learning fundamentals
- Build and train neural network models
- Prevent overfitting and improve model performance
- Apply deep learning to real-world datasets

## 📚 Topics Covered

- Introduction to Deep Learning
- Perceptron
- Artificial Neural Networks (ANN)
- Activation Functions
- Forward & Backpropagation
- Loss Functions
- Optimizers
- TensorFlow Basics
- Keras Sequential API
- Model Evaluation
