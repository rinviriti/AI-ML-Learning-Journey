# 🤖 Machine Learning

This module covers the fundamental concepts and practical implementation of Machine Learning using Scikit-Learn.

## 🎯 Learning Objectives

- Understand the Machine Learning workflow
- Prepare and preprocess datasets
- Train and evaluate models
- Build classification and regression systems

## 📚 Topics Covered

- Machine Learning Pipeline
- Train-Test Split
- Data Preprocessing
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- Model Evaluation
- Cross Validation

## 📁 Files

| File | Description |
|--------|-------------|
| ml_workflow.ipynb | Introduction to ML workflow |
| data_preprocessing.ipynb | Data cleaning and preprocessing |
| train_test_split.ipynb | Dataset splitting |
| linear_regression.ipynb | Regression example |
| logistic_regression.ipynb | Classification example |
| decision_tree.ipynb | Decision tree model |
| random_forest.ipynb | Random forest model |
| model_evaluation.ipynb | Accuracy and evaluation metrics |
