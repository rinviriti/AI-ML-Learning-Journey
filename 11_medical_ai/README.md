# 🩺 Medical AI

This module focuses on applying Artificial Intelligence and Deep Learning techniques to healthcare and medical imaging problems.

## 🎯 Learning Objectives

- Understand Medical AI workflows
- Learn medical image preprocessing
- Explore disease classification techniques
- Develop explainable healthcare AI systems

## 📚 Topics Covered

- Medical Image Analysis
- Image Preprocessing
- Medical Dataset Exploration
- CNN for Healthcare
- Explainable AI (XAI)
- Grad-CAM
- Disease Classification
- Model Evaluation
